发表于 2017-08-31 | 更新于 2019-12-31 | 评论数：

1.标题

标题分为一级标题，二级标题，以此类推到六级标题。#号后加空格再写文字。

示例：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

2.字体

加黑

加黑的字体用**包起来

斜体

斜体的字体用*包起来

斜体加粗

斜体加粗用3个*抱起来

删除线

删除线的字体用2个~~抱起来

示例：

**加黑**
*斜体*
***斜体加粗***
~~删除线的字体用2个~~抱起来
