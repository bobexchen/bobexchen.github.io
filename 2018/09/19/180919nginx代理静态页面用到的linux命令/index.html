nginx代理静态页面用到的linux命令

发表于 2018-09-19 | 更新于 2019-12-31 | 评论数：

sudo 以超级管理员做 super administrator
ls　　 显示文件或目录 ll 显示文件及详细信息
pwd 显示当前目录

service sshd status 查看Linux主机的ssh服务是否开启
service sshd status

cat /etc/group | grep username 在linux中查询一个用户属于哪些组

用户组的所有信息都存放在/etc/group文件中。

ps -ef|grep java 显示所有包含java的进程。

cat 查看文件内容

vi 文件名.txt :创建文件
esc :wq :保存文件
vi -R file 　:以只读模式启动vi

./ 启动war包

touch 文件名字 创建文件

mkdir 文件夹 创建文件夹

rm 删除空文件夹

rm -rf 删除文件夹和里面的东西

nginx -t nginx会去测试你得配置文件得语法，并告诉你配置文件是否写得正确，同时也告诉了你配置文件得路径

curl+网址 将网址的内容显示在屏幕上

sudo nginx -s reload 重启nginx

cp -r 文件名 要放的文件夹路径

ps -ef |grep nginx 找到进程号
kill -QUIT 进程号
关闭nginx

free -m 查看内存使用情况

sudo sysctl -w vm.drop_caches=3 释放部分内存

tail命令用于输入文件中的尾部内容。tail命令默认在屏幕上显示指定文件的末尾10行。

df df命令用于显示磁盘分区上的可使用的磁盘空间。默认显示单位为KB。可以利用该命令来获取硬盘被占用了多少空间，目前还剩下多少空间等信息。

wc wc命令用来计算数字。利用wc指令我们可以计算文件的Byte数、字数或是列数，若不指定文件名称，或是所给予的文件名为“-”，则wc指令会从标准输入设备读取数据。
